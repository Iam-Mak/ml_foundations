## Machine Learning Portfolio

A collection of machine learning projects and coursework covering core concepts from data preprocessing to advanced modeling techniques across supervised, unsupervised, and deep learning.

This work was completed as part of the Bertelsmann Tech Scholarship program on Udacity, awarded based on demonstrated performance and engagement.



### Supervised Machine Learning

Projects focused on prediction and classification tasks using labeled data.

* **Finding Donors for CharityML**
  Classification project to identify potential donors based on demographic and financial data.


### Deep Learning

Projects exploring neural networks and representation learning.

* **Image Classifier Project**
  Flower image classification using deep learning techniques.


### Unsupervised Machine Learning

Projects focused on discovering patterns in unlabeled data.

* **Customer Segmentation**
  Clustering techniques applied to identify distinct customer groups.
